
# portfolio-react

Home to my portfolio to showcase my work.



## Features

- Light/dark mode toggle
- Preview of my repositories
- Cross platform
- Contact Form using formspree


## Tech Stack
    React
    React Bootstrap
    Redux
    GitHub REST API
    FormSpree
